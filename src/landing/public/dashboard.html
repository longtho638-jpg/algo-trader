Quick Start Guide

Get your AlgoTrade bot running in 10 minutes

Choose Your Deployment

Apple Silicon Mac
M1 Max 64GB+ | MLX inference | $0/mo
☁
Cloud VPS
RTX 4090 24GB+ | Ollama | ~$150/mo
1

Install Dependencies

Homebrew, Node.js 20+, Python 3.11+, MLX LM

brew install node python@3.11 pip3 install mlx-lm
2

Download DeepSeek R1 Model (~18GB)

Start MLX server — model downloads automatically on first run

python3 -m mlx_lm.server \ --model mlx-community/DeepSeek-R1-Distill-Qwen-32B-4bit \ --port 11435 &
3

Clone & Build AlgoTrade

git clone https://github.com/longtho638-jpg/algo-trader.git cd algo-trader && npm install && npx tsc
4

Test with Dry Run

Replace KEY and SECRET with values from admin

node scripts/start-trading-bot.mjs \ --license-key=YOUR_KEY \ --secret=YOUR_SECRET \ --dry-run --capital=500
5

Go Live

Add your Polymarket wallet private key. Fund with USDC on Polygon.

node scripts/start-trading-bot.mjs \ --license-key=YOUR_KEY \ --secret=YOUR_SECRET \ --private-key=0xYOUR_WALLET_KEY \ --capital=500
6

Run as Background Service

Create a launchd plist to auto-start on boot. See full SOP in docs.

launchctl load ~/Library/LaunchAgents/com.algotrade.bot.plist tail -f /tmp/algotrade-bot.log
1

Provision GPU VPS

Install NVIDIA Drivers + Ollama

apt update && apt upgrade -y apt install -y nvidia-driver-535 curl -fsSL https://ollama.com/install.sh | sh ollama pull deepseek-r1:32b
3

Install Node.js + AlgoTrade

curl -fsSL https://deb.nodesource.com/setup_20.x | bash - apt install -y nodejs git clone https://github.com/longtho638-jpg/algo-trader.git cd algo-trader && npm install && npx tsc
4

Test with Dry Run

node scripts/start-trading-bot.mjs \ --license-key=YOUR_KEY \ --secret=YOUR_SECRET \ --dry-run --capital=500 \ --llm-url=http://127.0.0.1:11434/v1
5

Go Live + systemd Service

# Create /etc/systemd/system/algotrade.service # Then: systemctl enable algotrade systemctl start algotrade journalctl -u algotrade -f
